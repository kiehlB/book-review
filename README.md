This website is a platform for users to write book reviews, featuring only Korean books and supporting only the Korean language.

Website link: https://www.bookreview.pro/

# Project Stack
 
 * Next.js 
 * Typescript
 * Tailwind css
 * Redux Toolkit
 * Apollo GraphQL
 * TipTap
 * Framer Motion
